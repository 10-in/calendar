# 历法

## 安装使用

  ```shell
  composer require shiyin/calendar
  ```

## 功能

  - 公历转农历
  - 农历转公历
  - 干支纪年法
  - 属相
  - 星座
  - 节气

## 优化点
  - 对固定值结果不再重复计算，直接写公式计算的结果
